first aid

feel free to write a mail to the teaching assistant, if you have any problems / questions. he will be happy to help you out.

submission guidelines

always make sure to submit your files anonymized. for reports / evaluations it is recommended to use common filetypes like simple txt or pdf. if you would like to submit multiple files, please use a zip archive. it is advised to check your uploaded file by downloading it and verifying its correctness.
